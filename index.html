NIRA

NIRA-GUI allows solves coupled-constraint games. It implements an equilibrium search method based on the Nikaido-Isoda function and a relaxation algorithm, hence the acronym NIRA. This version of NIRA includes a graphical user interface which allows the user to specify the game symbolically.

Requirements

NIRA-GUI runs through MATLAB, and makes use of the Optimization and Symbolic Math toolboxes. It is tested on MATLAB 2013b.
