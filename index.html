NIRA

NIRA solves solve coupled-constraint games.

NIRA solves coupled-constraint games. It implements an equilibrium search method based on the Nikaido-Isoda function and a relaxation algorithm, hence the acronym NIRA. This version of NIRA includes a graphical user interface which allows the user to specify the game symbolically.

Version

The latest version is 1.1. Earlier versions were developed before NIRA was migrated to GitHub, they are available here.

Requirements

NIRA runs through MATLAB, and makes use of the Optimization and Symbolic Math toolboxes. It is tested on MATLAB 2013b.
